# 🌿 **RIKA 팀 소개**

> **Rika ⾥⾹ (마을의 향기)**  
> 시골(마을)에서 느끼는 정 많고 따뜻한 분위기로 게이머들을 환영하고,  
> 떠났다가 언제든 돌아오면 함께 쌓았던 즐거운 추억과 기억을 바탕으로  
> 예전의 향수를 느낄 수 있는 게임을 만드는 것을 목표로 합니다.

---

## 📌 **팀 개요**
- **설립일**: 2025.01.31 ~
- **비전**: 게이머들에게 따뜻함과 향수를 주는 게임 제작
- **키워드**: 감성, 커뮤니티, 향수, 즐거움
